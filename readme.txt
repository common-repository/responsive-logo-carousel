== Description ==
This plugin enables the user to create logo carousels in order to display maybe the companies, sponsors or clients they have worked or for other purposes. The carousel is responsive and can work on any device screen size.The carousel is based on the Owl carousel.

#### Plugin features: ####
* Fully responsive to all media screens.
* Supports all modern browsers.
* Compatible with all Wordpress themes.
* Easy to use.
* Lightweight.

#### How to use: ####
1. Click on the Responsive Logo Carousel menu item in the admin dashboard.
2. Click on Carousels sub menu and create a new caousel category by filling in the Name and saving it.
3. Click on the Add New sub menu and fill in the name of the Logo, choose the carousel on the right side where the logo will appear, set the logo as the featured image and save the logo.
4. You can repeat step 4 several times to add several logos to a carousel.
5. Add the logo carousel to a post by clicking on the the custom TinyMCE button which produces the shortcode ([responsive-logo-carousel category="logo-carousel-name"]).
6. Replace the "logo-carousel-name" with the name of the Logo Carousel created earlier and save the post in order to view the carousel.

== Frequently Asked Questions ==
How can I call the plugin in my posts?

Just use the shortcode: [responsive-logo-carousel category="logo-carousel-name"]
*Where 'logo-carousel-name' is the name of the logo carousel

Can I use the Responsive Logo Carousel on a page other than a post?
Yes, you can use the carousel even on a page.
